The **W**ater **Re**sources **S**imulation **L**anguage is the domain specific language used to develop [WRIMS](https://water.ca.gov/Library/Modeling-and-Analysis/Modeling-Platforms/Water-Resource-Integrated-Modeling-System) models (like [CalSim3](https://water.ca.gov/Library/Modeling-and-Analysis/Central-Valley-models-and-tools/CalSim-3))

> [!WARNING]
> This is the development repository for a pre-release version of the WRESL+ parser.
> Is you are wanting to submit an issue, or feature request for WRESL+, please do that through the [WRIMS 2 Repo](https://github.com/CentralValleyModeling/wrims).

Documentation is under development, and legecy documentation can be found on the [WRIMS DWR Page](https://water.ca.gov/Library/Modeling-and-Analysis/Modeling-Platforms/Water-Resource-Integrated-Modeling-System).
